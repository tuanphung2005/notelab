# notelab
A Tauri-based markdown note-taking app with a focus on performance and user experience.

## Recommended IDE Setup

## Development
```bash
bun i
bun run tauri android init
```

```bash
bun run tauri dev
```
or
```bash
bun run tauri android dev
```
